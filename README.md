# Vendas Pulse

O Vendas Pulse é um dashboard de vendas que eu desenvolvi para praticar HTML, CSS e JavaScript.

A ideia do projeto é simular um painel usado por uma loja para acompanhar pedidos, faturamento, produtos mais vendidos e desempenho das vendas.

## Objetivo do projeto

Meu objetivo com esse projeto foi criar uma aplicação mais completa do que uma landing page simples.

Eu quis praticar a construção de uma interface com cara de sistema real, usando dados, filtros, gráficos, cadastro de pedidos e armazenamento no navegador.

## Funcionalidades

- Cadastrar uma nova venda
- Editar uma venda já cadastrada
- Excluir uma venda
- Filtrar pedidos por cliente ou produto
- Filtrar pedidos por mês
- Filtrar pedidos por status
- Calcular faturamento automaticamente
- Calcular quantidade de pedidos
- Calcular ticket médio
- Calcular conversão de pedidos pagos
- Mostrar gráfico de faturamento mensal
- Mostrar ranking de produtos mais vendidos
- Exportar pedidos em CSV
- Limpar todos os dados
- Restaurar dados de exemplo
- Salvar os dados no navegador com localStorage
- Layout responsivo para desktop e celular

## Como usar

1. Abra o projeto no navegador.
2. Clique em **Nova venda** para cadastrar um pedido.
3. Use os filtros para buscar por cliente, produto, mês ou status.
4. Edite ou exclua pedidos pela tabela.
5. Use o botão **Exportar CSV** para baixar os pedidos filtrados.
6. Use **Limpar dados** para apagar os pedidos.
7. Use **Restaurar exemplo** para voltar aos dados iniciais.

## Como rodar o projeto

Baixe ou clone este repositório e abra o arquivo `index.html` no navegador.

Também é possível usar a extensão Live Server no VS Code.
